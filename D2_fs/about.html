About This Website

This website was created using HTML, CSS, and Node.js. The HTML provides the structure of the website, while CSS is used for styling and layout. Node.js, along with the 'fs' (file system) module, is used to manage the server-side operations and file handling.

The navigation bar at the top of the page is styled using CSS to provide a responsive and user-friendly interface. The logo is embedded directly into the HTML using a base64 encoded image.

Node.js is utilized to serve the HTML files and handle any backend logic required for the website. The 'fs' module is particularly useful for reading and writing files on the server, making it an essential part of the Node.js environment for this project.
